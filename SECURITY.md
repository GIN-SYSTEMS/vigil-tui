# Security Policy

## Supported Versions

Current security updates are provided for the following versions:

| Version | Supported |
| ------- | ----------- |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x: |

## Reporting a Vulnerability

**Please do not open a public issue for security vulnerabilities.**

Instead, please use the GitHub **Private Vulnerability Reporting** feature found under the "Security" tab of this repository. This allows us to discuss and fix the issue privately. 

I will acknowledge your report and provide a fix as soon as possible. Thank you for helping keep `vigil-tui` secure!
